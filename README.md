# Loja-de-tenis
Loja de tênis feita com ReactJS

Esse é um site de uma loja de tênis fictícia. Nela o usuário pode ver os produtos disponíveis. Os botões de compra, os elementos da navbar e o filtro não levam a novas páginas. São apenas elementos visuais. Pois a página foi feita focando no Front-End.
